Helyszín és év: Nyíregyháza, 2026

Projekt leírás: A projekt egy teljes stack webalkalmazás: Frontend:
React Backend: Spring Boot Adatbázis: MySQL (XAMPP)

Szükséges programok: - XAMPP - Node.js - Java JDK - Maven - Visual
Studio Code

Adatbázis (XAMPP): 1. XAMPP letöltése:
http://www.apachefriends.org/en/xampp.html

2.  Telepítés: C:`\xampp`{=tex}
3.  Indítás: Apache → Start MySQL → Start
4.  phpMyAdmin → Admin
5.  Adatbázis import (.sql)

Backend (Spring Boot): - Nyisd meg a backend mappát - Fontos fájl:
pom.xml

Indítás: mvn spring-boot:run

Dátumrendszer: Backend indítás után kell betölteni (projektfüggő).

Frontend (React): - Nyisd meg a Front mappát

Telepítés: npm install

Extra csomagok: npm install react-markdown remark-gfm

Indítás: npm start

Elérés: http://localhost:3000

Indítási sorrend: 
1. XAMPP indítás 
2. Apache + MySQL 
3. Adatbázis import
4. Backend indítás 
5. Frontend install + start 
6. Weboldal megnyitása

Projekt struktúra: /Front /backend /database
